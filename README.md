# My Portfolio Website

Welcome to my professional portfolio! This is a modern, responsive website showcasing my skills and projects.

## 🚀 Quick Start with Live Server

### Option 1: Using VS Code Live Server Extension (Easiest)

1. **Install Live Server Extension:**
   - Open VS Code
   - Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
   - Search for "Live Server"
   - Click Install (by Ritwick Dey)

2. **Start Live Server:**
   - Right-click on `index.html`
   - Select "Open with Live Server"
   - Your portfolio will open in your default browser at `http://127.0.0.1:5500`

3. **Auto-Reload:**
   - Any changes you make to HTML, CSS, or JS will automatically refresh in the browser

### Option 2: Using Python (Built-in)

**Python 3.x:**
```bash
python -m http.server 8000
```

**Python 2.x:**
```bash
python -m SimpleHTTPServer 8000
```

Then open: `http://localhost:8000`

### Option 3: Using Node.js (http-server)

1. **Install globally:**
   ```bash
   npm install -g http-server
   ```

2. **Start server:**
   ```bash
   http-server
   ```

Then open the provided local URL (usually `http://127.0.0.1:8080`)

### Option 4: Using Node.js (with npm)

1. **Install locally:**
   ```bash
   npm install http-server --save-dev
   ```

2. **Add to package.json scripts:**
   ```json
   "scripts": {
     "start": "http-server"
   }
   ```

3. **Run:**
   ```bash
   npm start
   ```
